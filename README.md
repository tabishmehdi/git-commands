# Git Commands List

A collection of basic Git commands with explanations.

## Basic Git Commands

1. **`git init`**  
   Initialize a new Git repository in your project folder.

2. **`git clone <url>`**  
   Clone a remote repository from a given URL to your local machine.

3. **`git status`**  
   Show the current state of the working directory and staging area.

4. **`git add <file>`**  
   Stage changes (add files to the staging area) for the next commit.

5. **`git commit -m "message"`**  
   Commit staged changes with a descriptive message.

6. **`git push`**  
   Upload local commits to a remote repository.

7. **`git pull`**  
   Fetch and integrate changes from the remote repository into your local branch.

8. **`git branch`**  
   List, create, or delete branches in your repository.

9. **`git checkout <branch>`**  
   Switch to a different branch or restore working tree files.

10. **`git merge <branch>`**  
    Merge changes from one branch into your current branch.
